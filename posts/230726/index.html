Blender 快捷键

MiSaKa89122023-07-262023-12-13

基础常用

按键 / 组合键 功能
鼠标中键 拖动 视图旋转
SHIFT + 鼠标中键 拖动 视图移动
CTRL + 鼠标中键 拖动 / 鼠标滚轮 视图缩放
（ALT + ）G *1（清除）物体移动
（ALT + ）S *1（清除）物体缩放
（ALT + ）R *1（清除）物体旋转
SHIFT + 鼠标右键 快速将游标设置在指针的位置
SHIFT + A 添加物体菜单
X 删除选中的物体
TAB 进入点线面的编辑模式
SHIFT + D 复制物体
/ 进入隔离模式进行编辑
N 鼠标指针所在窗口的操作面板
ALT + 鼠标左键 *2 循环选择
  1. *1 进入对应模式后按 XYZ 可 指定要调整的轴向，随后 输入数字 可以精确调整想要的数值
  2. *1 进入对应模式后按 SHIFT + XYZ 可 忽略指定轴向的调整，如果输入数字同上
  3. *1 进入对应模式后按 鼠标右键 可 取消未确认的操作
  4. *2 选择时会根据 鼠标指针所在的位置 自动选择 竖向或横向 的环选方式
  5. *2 按住 SHIFT 时选择可 加选或减选

工具与物体

按键 / 组合键 功能
W 选择
D 标注
E 挤出选区
I 内插面
CTRL + B 倒角
CTRL + R 环切
K 切割
A 全选
A + A 取消选择
CTRL + I 反向选择
CTRL + J 合并物体
按键 / 组合键 功能
H 隐藏物体
ALT + H 取消隐藏物体
SHIFT + H 隐藏除选择外的物体
CTRL + . 进入物体原点的编辑模式
F 从边创建面
SHIFT + N 重新计算法线
ALT + A 物体对齐

菜单与饼菜单

按键 / 组合键 功能
SHIFT + A 添加新的物体和内容
M 将选中的物体移动到合集
CTRL + A 应用变换
CTRL + L 关联/传递数据
I 添加关键帧
按键 / 组合键 功能
~ 视图切换
TAB 编辑模式
SHIFT + S 游标相关
, 坐标轴变换
. 轴心点变换
鼠标按键 5 视图相关
鼠标按键 4 着色相关

动画

按键 / 组合键 功能
I 添加关键帧

视图与摄像机

按键 / 组合键 功能
鼠标按键 5 视图饼菜单
ALT + Z 透写模式
SHIFT + Z 线框模式
按键 / 组合键 功能
0 进入或退出摄像机视角
CTRL + ALT + 0 将摄像机视角定位到当前的自由视角

雕刻

按键 / 组合键 功能
F 调整笔刷大小
SHIFT + F 调整笔刷强度
R 调节体素大小网格
按住 SHIFT + 笔刷 平滑雕刻
按住 CTRAL + 笔刷 反向雕刻
